HttpSMS Integration

HttpSMS uses your Android phone as a message gateway.
Get your API key at httpsms.com → Settings → API. The Sender Number is the Android device's number (e.g. +9779845xxxxxx).

Green API — WhatsApp Group (Auto Send)

One-time setup:
1. Register at green-api.com → create a free instance.
2. Copy Instance ID, API Token, and API URL from the instance panel.
3. Paste below → click "Check Status / Show QR" → scan with WhatsApp.
4. Get group chatId: console → API → getChats → Execute → copy group id (e.g. 120363xxxxxx@g.us).

Scan with WhatsApp → Settings → Linked Devices → Link a Device

QR expires ~20s — click Refresh QR if faded.
